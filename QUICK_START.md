# 🚀 Quick Start Guide

## Getting Started in 3 Steps

### 1. Open the Game
- Double-click `index.html` or open it in your browser
- Works in Chrome, Firefox, Safari, or Edge

### 2. Choose a Game
All four games are ready to play:
- **Top-Left**: Tic Tac Toe - Click cells to play
- **Top-Right**: Snake - Click "Start Game" and use arrow keys
- **Bottom-Left**: 8 Queens - Click cells to place queens
- **Bottom-Right**: Scoreboard - Tracks all scores automatically

### 3. Start Playing!
- Player 1 (Blue) starts first
- Take turns playing any game
- Watch your scores update in real-time

## 🎮 Quick Controls

### Tic Tac Toe
- **Click** empty cells to place X or O
- **Reset** button to start new game

### Snake Game
- **Start Game** button to begin
- **Arrow Keys** (↑↓←→) to move
- **Pause** button to pause/resume
- 60 seconds per turn

### 8 Queens Puzzle
- **Click** cells to place queens
- Orange = cells under attack (invalid)
- **Reset** button to clear board

### Scoreboard
- **Reset All** button to clear all scores
- Activity log shows recent events

## 💡 Pro Tips

1. **Tic Tac Toe**: Control the center square
2. **Snake**: Use edges, don't trap yourself
3. **8 Queens**: Plan ahead, watch attack patterns
4. **Strategy**: Balance all games for highest total score

## 🎯 Winning

- Play multiple rounds
- Accumulate points across all games
- Highest total score wins!
- Have fun! 🎉
